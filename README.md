
# MobileFoodFreshness (Android + CameraX + TFLite + OpenCV)

https://github.com/json0419-cell/computervision-Project2
## Step 1: Download and unzip

Go to the repo and select Download Zip.

<img src="https://github.com/json0419-cell/computervision-Project2/blob/main/fig1.png" style="width: 50%; height: 50%"/>

## Step 2: Unzip the file

Unzip with your unarchiver of choice into a folder of the same name. This can be in a “temp” directory or anywhere else. In these instructions, I have placed the folder in “Downloads”.

<img src="https://github.com/json0419-cell/computervision-Project2/blob/main/fig2.png" style="width: 50%; height: 50%"/>

## Step 3: Check directory structure. 

The resulting directory structure looks like this:

<img src="https://github.com/json0419-cell/computervision-Project2/blob/main/fig3.png" style="width: 50%; height: 50%"/>

## Step 4: Open the project

In Android Studio, open the project, selecting the unzipped folder.

<img src="https://github.com/json0419-cell/computervision-Project2/blob/main/fig4.png" style="width: 50%; height: 50%"/>
<img src="https://github.com/json0419-cell/computervision-Project2/blob/main/fig5.png" style="width: 50%; height: 50%"/>

## Step 5: Edit local.properties to point to the Android SDK on your computer. 

Here is an example of what Mac OS looks like:

sdk.dir=/Users/sambartleman/Library/Android/sdk/

On Windows this might look like:

sdk.dir=C\:\\Users\\brans\\AppData\\Local\\Android\\Sdk

## Step 6: Build and Run

You should now be able to set the build target to “app” and select the target you wish, either an emulator or local device. Clicking the green run button in Android Studio will launch the app. (Note you may also need to sync gradle in order to build dependencies.)
